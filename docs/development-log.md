# 개발 과정 기록

## Day 1 (2024.09.24)

### 오전 - 환경 설정 및 아키텍처 설계

- VSCode 개발 환경 구축 완료
- Docker Compose를 이용한 MySQL Master-Slave 구성 결정
- AsyncIO 기반 비동기 처리 구조 설계

**주요 의사결정**:

- Python AsyncIO vs Threading: 동시성 처리 효율성으로 AsyncIO 선택
- MySQL 8.0 선택: GTID 기반 복제와 성능 개선사항 활용

### 오후 - 성능 테스트 모듈 구현

- Connection Pool 최적화 이슈 발생 및 해결
- QPS 측정 로직 구현 (실시간 메트릭 수집)
- P95, P99 응답시간 계산 알고리즘 구현

**발생한 문제**:

- 초기 Connection Pool 설정으로 50 QPS만 달성
- maxsize=20, minsize=5로 조정 후 199.4 QPS 달성

### 오전 - 일관성 및 안정성 테스트 구현

- 전체 테스트 실행 시간 10분 15초로 최적화
